## 使用root账户

1. 在实际生产操作中，我们基本上都是使用超级管理员账户操作Linux系统，也就root用户；Linux系统默认是关闭root用户的，我们需要为root用户设置一个初始化密码以方便我们使用。
    + 设置root账户的密码:
        ``` shell 
        sudo passwd root 
        ```
    + 切换到root:
        ``` shell 
        su
        ```
    + 设置运行远程登录root:
        ``` shell
         vim /etc/ssh/sshd_config
         
         # Authentication:
         LoginGraceTime 120
         # PermitRootLogin prohibit-password  # 注释此行
         PermitRootLogin yes # 添加此行
         StrictModes yes
         
         重启ssh服务
         service ssh restart
        ```
